# 小宇宙下载助手离线安装说明

适用场景：无法正常访问 Chrome Web Store，需要手动安装扩展。

## 安装步骤

1. 下载并解压 `xyzdownloader-v1.2.0.zip`
2. 打开 Chrome，在地址栏输入 `chrome://extensions`
3. 打开右上角“开发者模式”
4. 点击左上角“加载已解压的扩展程序”
5. 选择刚刚解压后的文件夹

选择文件夹时，请直接选择包含 `manifest.json` 的那个目录。

安装成功后，扩展列表里会显示：

`小宇宙下载助手 1.2.0`

## 使用方法

1. 打开任意小宇宙单集页面，例如：
   `https://www.xiaoyuzhoufm.com/episode/6960e332e235ea65bcf89e8d`
2. 在节目封面上方找到“下载音频”按钮
3. 点击按钮后，浏览器会开始下载对应音频文件

## 常见问题

### 1. 为什么不能直接双击安装 `.crx`？

新版 Chrome 对非 Chrome Web Store 来源的 `.crx` 安装限制较多，很多环境下会直接拦截或强提示。

因此离线安装时，推荐使用“加载已解压的扩展程序”，这是目前更稳定的方式。

### 2. 如果扩展更新了，怎么升级？

1. 下载新的离线安装包并解压
2. 打开 `chrome://extensions`
3. 先删除旧版本，或者重新选择新的解压目录加载

### 3. 如果按钮没有显示怎么办？

1. 确认页面是小宇宙单集页面，地址形如 `/episode/...`
2. 刷新页面重试
3. 在 `chrome://extensions` 中确认扩展已启用
